== Quotes on Rails

Quotes on Rails is a quote database written with the [Ruby on Rails]{http://www.rubyonrails.org} web framework. People can read and vote on hilarious quotes from IRC, and submit their own.

== Install

* Install Rails 2.1.1 (sudo gem install rails -v 2.1.1)
* Download and unzip Quotes on Rails
* Edit the database.yml and the settings.yml files
* Rake db:migrate
* Have fun :)
